# Responsive E-Commerce Homepage

This project is a fully responsive e-commerce homepage built using **HTML, CSS, and JavaScript**, developed as part of a frontend assignment.

##  Tech Stack
- HTML5
- CSS3 
- JavaScript 

## Features
- Pixel-accurate design matching provided desktop and mobile layouts
- Fully responsive across desktop, tablet, and mobile devices
- Clean and modular code structure
- Interactive UI elements using JavaScript
- Optimized image usage and layout consistency

index.html
style.css
script.js
images/
